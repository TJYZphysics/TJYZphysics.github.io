# TJYZ Physics 社团官网设计规格

## 目标

为 TJYZ Physics 构建并部署一个纯前端官网，包含主页、博客、实验与 About us 四个模块。网站应具有明确的科学气质、成熟的视觉表达、可靠的响应式布局，并允许维护者仅通过新增 Markdown 文件发布内容。

## 技术架构

- 使用 React、TypeScript 与 Vite 构建单页应用，通过 React Router 提供稳定路由。
- 博客正文放置在 `blog/`，社团介绍与历史放置在 `about/`。
- Vite 构建阶段使用 `import.meta.glob` 扫描 Markdown；新增文件无需手工维护索引。
- Markdown 以 UTF-8 读取，使用 gray-matter 解析 front matter，react-markdown 渲染正文，并对外链、代码、表格和图片提供安全样式。
- 三个互动实验均在浏览器端运行，不依赖后端。
- GitHub Actions 构建 `dist/` 并部署至 GitHub Pages。

## 视觉系统

### 主页、博客与实验

主色为午夜靛蓝 `#080B1A`，辅以光谱青 `#59E1FF`、电光蓝 `#6487FF` 和暖白 `#F3F5F8`。界面保持低饱和深色基底，以物理轨迹、网格、光晕与精密仪器材质建立科学感，避免俗套的紫色 SaaS 渐变和卡通科学图标。

主页采用非对称编辑式构图。首屏只保留社团名称、简短主张和两个明确入口；背景 Canvas 粒子轨道响应指针但不干扰阅读。后续内容以观测、实验、分享三个行为和精选内容组成，避免长篇宣传文案。

博客采用高级拟态笔记本风格。文章列表位于有纸张层次、装订线与微妙纹理的阅读表面中；正文保持高对比、舒适行宽和稳定行高。移动端去除非必要装订装饰，保证内容优先。

实验采用仪器面板式拟物设计，并叠加克制的 Web 玻璃材质近似效果。控制区和画布区在桌面端并排，在窄屏纵向排列；所有按钮、滑杆与状态反馈均有清晰层级和键盘焦点样式。

### About us

About us 使用独立的瑞士国际主义主题：暖米色 `#EDE5D4`、信号红 `#E63323` 与深黑 `#141414`。页面采用严格网格、粗体无衬线标题、强对齐和档案式时间轴。介绍和历史均来自 Markdown，并禁止出现任何成员姓名或可识别个人信息。

## 内容与数据流

博客 Markdown front matter 支持 `title`、`date`、`summary`、`tags`、`cover`。构建时自动生成文章集合，按日期倒序展示；缺少可选字段时提供稳定回退，缺少标题时使用文件名。无文章时显示明确的空状态。

About us 固定加载 `about/introduction.md` 与 `about/history.md`。内容保持组织层面的描述，只呈现社团使命、活动形式、发展阶段与未来方向。

点赞使用浏览器 localStorage 保存每篇内容的本地状态并提供即时反馈。评论区域预留 Giscus 接口；未配置仓库参数时显示说明性占位，不造成报错。

## 实验设计

### 三体模拟器

基于二维牛顿引力积分。用户可调整三个天体的质量、时间尺度和轨迹长度，选择预设、暂停、继续、单步和重置。画布显示天体、尾迹、质心与运行状态。数值异常时自动恢复预设并给出提示。

### 碰撞模拟器

一维碰撞演示两个物体的动量传递。用户可调整质量、初速度和恢复系数，启动、暂停与重置。界面实时展示碰撞前后速度、总动量和动能变化，并在碰撞完成后形成闭环结果说明。

### 原创游戏：因果折纸

玩家在离散时空网格上放置有限数量的“曲率折点”，改变信号下一步允许的传播方向，使两束从不同事件出发的信号在目标事件相遇，同时不得穿出光锥或进入禁区。关卡提供放置、撤销、清空、运行和重新开始操作，显示因果合法性、剩余折点与结果反馈。这是为本站原创设计的玩法组合，不宣称不存在任何宽泛概念上的相似作品。

## 响应式与无障碍

- 支持 360px 至宽屏桌面布局，任何页面不得横向溢出。
- 正文行高保持在 1.65 至 1.8，标题使用紧凑但不相撞的行高。
- 导航在移动端折叠为可关闭菜单；路由变化后恢复顶部并关闭菜单。
- Canvas 提供文本替代说明；交互控件具备标签、焦点态和合理触控尺寸。
- 尊重 `prefers-reduced-motion`，降低粒子、转场和光晕动画。

## 验收与发布

- 单元测试覆盖 Markdown 元数据、物理计算、碰撞守恒、因果路径规则和本地点赞状态。
- 构建过程必须无 TypeScript 错误。
- 使用真实浏览器手动检查桌面与移动尺寸，逐一操作三个实验，并通过截图检查遮挡、文字重叠、间距和视觉一致性。
- Git 提交作者设置为 `TJYZphysics`，通过 GitHub 插件将最终提交更新至默认分支并确认 Pages 部署状态。
